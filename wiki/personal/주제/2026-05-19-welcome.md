# 환영합니다

이 노드는 viewer가 빈 그래프를 보여주지 않도록 넣어둔 시작 노드입니다.

## 첫 시작 단계

### 1. 첫 노드 만들기

**Claude로 (가장 빠름)**:
```bash
cd <이 폴더>
claude
> /wiki-ingest https://your-favorite-article.com
```

**직접 작성**:
```bash
cat > wiki/personal/통찰/$(date +%F)-first-insight.md <<'EOF'
---
id: 2026-XX-XX-first-insight
title: 첫 통찰
node_type: 통찰
memory_type: mental_model
created: 2026-XX-XX
last_reviewed: 2026-XX-XX
links: []
---

# 첫 통찰

여기 본문.
EOF
```

### 2. 이 welcome 노드 지우기

```bash
rm wiki/personal/주제/2026-05-19-welcome.md
```

### 3. 로컬에서 그래프 확인

```bash
cd viewer && npm install && npm run dev
# http://localhost:3000
```

### 4. push → Vercel 자동 배포

```bash
git add wiki/ && git commit -m "first node"
git push
```

## 자세한 사용법

루트의 `README.md`와 `CLAUDE.md`를 참고하세요.

## 스키마

이 노드는 `주제(topic)` 타입이고 `mental_model` 메모리 타입입니다. 노드 타입 6종(의미·통찰·절차·사건·주장·주제)과 엣지 타입 9종은 `CLAUDE.md`에 정의되어 있습니다.
